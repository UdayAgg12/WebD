Understanding CSS: External, Internal & Inline

A compact example showing how to use each method and how specificity works.

External CSS

This card is primarily styled by the styles.css external stylesheet.

External styles are ideal for site-wide styles and reuse.

Internal CSS

This card shows styles defined in the document's <style> block (inside the head).

Internal CSS is useful for single-page customizations or small demos.

This inline-styled paragraph overrides color from other styles.

Specificity & Cascade

Order of precedence from lowest to highest: browser defaults → external → internal → inline. Specificity and the cascade can change that order.

Try editing styles.css and the <style> block to see which rule wins.

Quick Examples

  • External file: styles.css — layout, base colors, typography
  • Internal: Page-specific tweaks inside <style> in the head
  • Inline: style="..." applied directly on an element for one-off overrides
